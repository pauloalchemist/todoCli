## TodoCli With Go

A simple command line tool (cli) for managing tasks.

In this project is exercised:

- Get input from users
- Go modules
- Go packages
- Tests

```zsh
Usage of ./todocli:
  -complete int
    	Alterar tarefa para concluida
  -list
    	Listar todas as tarefas
  -task string
    	Tarefa a ser inclusa no ToDo List
```
